# WhatsApp Instant Chat

> Whatsapp Instant Chat with Vuejs

## Install

```bash
git clone https://github.com/edytk/Whatsapp-Instant-Chat-with-Vuejs
cd app-whatsapp-url/
yarn install
yarn serve
```

open [localhost:8080](http://localhost:8080/)
